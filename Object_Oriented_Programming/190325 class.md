* 구조체
1. 클래스와 동일하지만, 디폴트가 public이다. (클래스는 디폴트가 private다.)

<code>

    struct Circle {
      Circle();
    private:
      int radius;
    };
    
</code>

<code>

    class Circle{
      int radius;
    public:
      Circle();
    }
    
</code>

위의 두 개는 완전히 같은 역할을 한다. </br>
